# Project Summary

## Objective

Build an end-to-end customer churn analytics platform capable of identifying at-risk customers and estimating revenue exposure.

## Dataset

- Customers: 10,000
- Payments: 100,000
- Usage Logs: 300,000
- Support Tickets: 50,000
- Total Records: 480,000+

## Key Results

- Churn Rate: 20.69%
- Revenue Generated: ₹80M+
- Revenue from Churned Customers: ₹16M+
- Customer Risk Classification Implemented

## Technologies

- PostgreSQL
- SQL
- Tableau Public
- GitHub

## Deliverables

- Relational Database Schema
- Data Generation Scripts
- Analytics Views
- KPI Reports
- Tableau Dashboard
- Business Case Study
